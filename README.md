# React Simple Boilerplate

> Simple boilerplate for react applications

This boilerplate contains:

- React 16
- React Router
- Styled Components
- Axios
- Lodash
- Jest

## How to use

- Clone this **repo** and enter it

- Install the dependencies

```
yarn
```

## Commands available

- To run the app use

```
yarn dev
```

- To run the tests use

```
yarn test
```

- To run the tests in watch mode use

```
yarn test:watch
```
